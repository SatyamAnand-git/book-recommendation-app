# Book Recommendation App

A full-stack book recommendation platform built with Spring Boot, MongoDB, JWT Authentication, Next.js and TypeScript.

## Features
- User Signup/Login
- JWT Authentication
- Secure Routes
- Add Books
- Edit Books
- Delete Books
- Search Books
- Genre Filtering

## Tech Stack
- Spring Boot
- Spring Security
- MongoDB
- JWT
- Next.js
- TypeScript
- Tailwind CSS
